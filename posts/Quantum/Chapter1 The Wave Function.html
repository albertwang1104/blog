Chapter 1. The Wave Function

薛丁格方程式 (The Schrödinger Equation)

在古典力學中，目標是透過牛頓第二定律（F=maF=maF=ma）配合初始條件來確定粒子的位置 x(t)x(t)x(t)。量子力學則採取完全不同的方法，尋找粒子的波函數（Wave function） Ψ(x,t)\Psi(x, t)Ψ(x,t)，並透過解薛丁格方程式來獲得：

iℏ∂Ψ∂t=−ℏ22m∂2Ψ∂x2+VΨi\hbar \frac{\partial \Psi}{\partial t} = -\frac{\hbar^2}{2m} \frac{\partial^2 \Psi}{\partial x^2} + V\Psi iℏ∂t∂Ψ​=−2mℏ2​∂x2∂2Ψ​+VΨ

其中 iii 是虛數單位，ℏ\hbarℏ 是約化普朗克常數（h/2π≈1.05×10−34 J sh/2\pi \approx 1.05 \times 10^{-34} \text{ J s}h/2π≈1.05×10−34 J s）。薛丁格方程式在邏輯上類比於牛頓第二定律，只要給定初始條件 Ψ(x,0)\Psi(x, 0)Ψ(x,0)，它就能決定未來任何時間的波函數 Ψ(x,t)\Psi(x, t)Ψ(x,t)。

統計詮釋 (The Statistical Interpretation)

波函數 Ψ\PsiΨ 本身是擴散在空間中的函數，但粒子在本質上是點狀的。根據波恩（Born）的統計詮釋，波函數絕對值的平方 ∣Ψ(x,t)∣2|\Psi(x, t)|^2∣Ψ(x,t)∣2 代表機率密度。具體而言，在時間 ttt 於 xxx 到 x+dxx+dxx+dx 之間找到粒子的機率為：

∣Ψ(x,t)∣2dx|\Psi(x, t)|^2 dx ∣Ψ(x,t)∣2dx

這引入了量子力學的不確定性（Indeterminacy）。對於測量前的粒子狀態，物理學界主要有三種觀點：

  1. 實在論（Realist）：認為粒子原本就在某處，量子力學是不完備的，需要「隱變量」來描述。
  2. 正統觀點（Orthodox）：又稱哥本哈根詮釋，認為測量前粒子不在任何地方，是測量行為強迫粒子「表態」。
  3. 不可知論（Agnostic）：拒絕回答測量前的狀態。
    貝爾定理（Bell's theorem）之後的實驗已證實正統觀點是正確的。此外，測量行為會導致波函數塌縮（Collapse of the wave function），使波函數瞬間變為一個尖峰。

機率 (Probability)

量子力學廣泛應用機率理論。以下是核心概念：

  • 期望值（Expectation Value）：標記為 ⟨j⟩\langle j \rangle⟨j⟩，代表對大量完全相同的系統進行測量所得結果的平均值，而非單次測量最可能的結果。
  • 標準差（Standard Deviation）：標記為 σ\sigmaσ，用來衡量機率分布的「擴散」程度。其計算公式為 σ2=⟨j2⟩−⟨j⟩2\sigma^2 = \langle j^2 \rangle - \langle j \rangle^2σ2=⟨j2⟩−⟨j⟩2。
  • 連續分布：引入機率密度 ρ(x)\rho(x)ρ(x)，在區間 aaa 到 bbb 找到粒子的機率為 Pab=∫abρ(x)dxP_{ab} = \int_a^b \rho(x) dxPab​=∫ab​ρ(x)dx。

歸一化 (Normalization)

由於粒子在空間中一定存在於某處，因此在全空間找到粒子的總機率必須等於 1，這稱為歸一化條件：

∫−∞+∞∣Ψ(x,t)∣2dx=1\int_{-\infty}^{+\infty} |\Psi(x, t)|^2 dx = 1 ∫−∞+∞​∣Ψ(x,t)∣2dx=1

假設我在時間 t=0t = 0t=0 時將波函數歸一化了。我該如何知道隨著時間流逝和 Ψ\PsiΨ 的演化，它會保持歸一化狀態？幸運的是，薛丁格方程式具有一個性質，即它會自動保留波函數的歸一化——如果沒有這個關鍵特徵，薛丁格方程式將與統計詮釋不相容，整個理論也會瓦解。 所以我們接下來仔細證明這一點：

ddt∫−∞+∞∣Ψ(x,t)∣2dx=∫−∞+∞∂∂t∣Ψ(x,t)∣2dx\frac{d}{dt} \int_{-\infty}^{+\infty} |\Psi(x, t)|^2 dx = \int_{-\infty}^{+\infty} \frac{\partial}{\partial t} |\Psi(x, t)|^2 dx dtd​∫−∞+∞​∣Ψ(x,t)∣2dx=∫−∞+∞​∂t∂​∣Ψ(x,t)∣2dx

注意，積分僅是 ttt 的函數，所以在第一項使用全導數 (d/dtd/dtd/dt)，但被積函數同時是 xxx 與 ttt 的函數，所以在第二項使用偏導數 (∂/∂t\partial/\partial t∂/∂t)。

根據乘積法則：

∂∂t∣Ψ∣2=∂∂t(Ψ∗Ψ)=Ψ∗∂Ψ∂t+∂Ψ∗∂tΨ\frac{\partial}{\partial t} |\Psi|^2 = \frac{\partial}{\partial t} (\Psi^* \Psi) = \Psi^* \frac{\partial \Psi}{\partial t} + \frac{\partial \Psi^*}{\partial t} \Psi ∂t∂​∣Ψ∣2=∂t∂​(Ψ∗Ψ)=Ψ∗∂t∂Ψ​+∂t∂Ψ∗​Ψ

現在薛丁格方程式說明：

∂Ψ∂t=iℏ2m∂2Ψ∂x2−iℏVΨ\frac{\partial \Psi}{\partial t} = \frac{i\hbar}{2m} \frac{\partial^2 \Psi}{\partial x^2} - \frac{i}{\hbar} V\Psi ∂t∂Ψ​=2miℏ​∂x2∂2Ψ​−ℏi​VΨ

以及其共軛複數：

∂Ψ∗∂t=−iℏ2m∂2Ψ∗∂x2+iℏVΨ∗\frac{\partial \Psi^*}{\partial t} = -\frac{i\hbar}{2m} \frac{\partial^2 \Psi^*}{\partial x^2} + \frac{i}{\hbar} V\Psi^* ∂t∂Ψ∗​=−2miℏ​∂x2∂2Ψ∗​+ℏi​VΨ∗

因此：

∂∂t∣Ψ∣2=iℏ2m(Ψ∗∂2Ψ∂x2−∂2Ψ∗∂x2Ψ)=∂∂x[iℏ2m(Ψ∗∂Ψ∂x−∂Ψ∗∂xΨ)]\frac{\partial}{\partial t} |\Psi|^2 = \frac{i\hbar}{2m} \left( \Psi^* \frac{\partial^2 \Psi}{\partial x^2} - \frac{\partial^2 \Psi^*}{\partial x^2} \Psi \right) = \frac{\partial}{\partial x} \left[ \frac{i\hbar}{2m} \left( \Psi^* \frac{\partial \Psi}{\partial x} - \frac{\partial \Psi^*}{\partial x} \Psi \right) \right] ∂t∂​∣Ψ∣2=2miℏ​(Ψ∗∂x2∂2Ψ​−∂x2∂2Ψ∗​Ψ)=∂x∂​[2miℏ​(Ψ∗∂x∂Ψ​−∂x∂Ψ∗​Ψ)]

積分現在可以顯式計算：

ddt∫−∞+∞∣Ψ(x,t)∣2dx=iℏ2m(Ψ∗∂Ψ∂x−∂Ψ∗∂xΨ)∣−∞+∞\frac{d}{dt} \int_{-\infty}^{+\infty} |\Psi(x, t)|^2 dx = \frac{i\hbar}{2m} \left( \Psi^* \frac{\partial \Psi}{\partial x} - \frac{\partial \Psi^*}{\partial x} \Psi \right) \bigg|_{-\infty}^{+\infty} dtd​∫−∞+∞​∣Ψ(x,t)∣2dx=2miℏ​(Ψ∗∂x∂Ψ​−∂x∂Ψ∗​Ψ)​−∞+∞​

但當 xxx 趨於 (±\pm±) 無窮大時，Ψ(x,t)\Psi(x, t)Ψ(x,t) 必須趨於零——否則波函數將無法歸一化。 由此可知：

ddt∫−∞+∞∣Ψ(x,t)∣2dx=0\frac{d}{dt} \int_{-\infty}^{+\infty} |\Psi(x, t)|^2 dx = 0 dtd​∫−∞+∞​∣Ψ(x,t)∣2dx=0

Info

因此可知薛丁格方程式具有一項關鍵特性：如果波函數在 t=0t=0t=0 時已歸一化，則方程式演化會自動保持歸一化狀態隨時間不變。數學上可滿足的狀態必須是 平方可積（Square-integrable） 的函數。

Momentum

在量子力學中，位置期望值 ⟨x⟩\langle x \rangle⟨x⟩ 的物理意義並非對「單一粒子」進行重複測量的平均值。在測量後將會使波函數塌縮（collapse）到實際測得的那個數值上的一個尖峰，第一次測量會使波函數塌縮到某個確定的數值，隨後的重複測量只會得到相同的結果。實際上，⟨x⟩\langle x \rangle⟨x⟩ 對一整套系綜（Ensemble）——即無數個處於相同狀態 Ψ\PsiΨ 的獨立系統——同時進行測量所得結果的統計平均值。

我們從位置期望值隨時間的變化率 d⟨x⟩/dtd\langle x \rangle / dtd⟨x⟩/dt 開始推導動量算符的表現形式。

首先，位置期望值為

⟨x⟩=∫−∞+∞x∣Ψ∣2 dx\langle x \rangle = \int_{-\infty}^{+\infty} x |\Psi|^2 \, dx ⟨x⟩=∫−∞+∞​x∣Ψ∣2dx

根據定義，位置隨時間的變化率為

d⟨x⟩dt=ddt∫x∣Ψ∣2 dx=∫x∂∂t∣Ψ∣2 dx\frac{d\langle x \rangle}{dt} = \frac{d}{dt} \int x |\Psi|^2 \, dx = \int x \frac{\partial}{\partial t} |\Psi|^2 \, dx \quad \text{} dtd⟨x⟩​=dtd​∫x∣Ψ∣2dx=∫x∂t∂​∣Ψ∣2dx

根據前面歸一化中的證明可得

d⟨x⟩dt=iℏ2m∫x∂∂x(Ψ∗∂Ψ∂x−∂Ψ∗∂xΨ)dx\frac{d\langle x \rangle}{dt} = \frac{i\hbar}{2m} \int x \frac{\partial}{\partial x} \left( \Psi^* \frac{\partial \Psi}{\partial x} - \frac{\partial \Psi^*}{\partial x} \Psi \right) dx \quad \text{} dtd⟨x⟩​=2miℏ​∫x∂x∂​(Ψ∗∂x∂Ψ​−∂x∂Ψ∗​Ψ)dx

利用分部積分法 ∫u dv=uv−∫v du\int u \, dv = uv - \int v \, du∫udv=uv−∫vdu，令 u=xu = xu=x，dv=∂∂x(… )dxdv = \frac{\partial}{\partial x} (\dots) dxdv=∂x∂​(…)dx。由於 Ψ\PsiΨ 在無窮遠處趨近於 0，邊界項消失：

d⟨x⟩dt=−iℏ2m∫(Ψ∗∂Ψ∂x−∂Ψ∗∂xΨ)dx\frac{d\langle x \rangle}{dt} = -\frac{i\hbar}{2m} \int \left( \Psi^* \frac{\partial \Psi}{\partial x} - \frac{\partial \Psi^*}{\partial x} \Psi \right) dx \quad \text{} dtd⟨x⟩​=−2miℏ​∫(Ψ∗∂x∂Ψ​−∂x∂Ψ∗​Ψ)dx

針對被積函數中的第二項 −∫∂Ψ∗∂xΨ dx-\int \frac{\partial \Psi^*}{\partial x} \Psi \, dx−∫∂x∂Ψ∗​Ψdx，再次使用分部積分（將導數從 Ψ∗\Psi^*Ψ∗ 轉移到 Ψ\PsiΨ）：

−∫∂Ψ∗∂xΨ dx=∫Ψ∗∂Ψ∂x dx-\int \frac{\partial \Psi^*}{\partial x} \Psi \, dx = \int \Psi^* \frac{\partial \Psi}{\partial x} \, dx \quad \text{} −∫∂x∂Ψ∗​Ψdx=∫Ψ∗∂x∂Ψ​dx

將此結果代回原式，括號內的兩項變為相同：

d⟨x⟩dt=−iℏ2m(2∫Ψ∗∂Ψ∂xdx)=ℏim∫Ψ∗∂Ψ∂xdx\frac{d\langle x \rangle}{dt} = -\frac{i\hbar}{2m} \left( 2 \int \Psi^* \frac{\partial \Psi}{\partial x} dx \right) = \frac{\hbar}{im} \int \Psi^* \frac{\partial \Psi}{\partial x} dx \quad \text{} dtd⟨x⟩​=−2miℏ​(2∫Ψ∗∂x∂Ψ​dx)=imℏ​∫Ψ∗∂x∂Ψ​dx

定義動量期望值

假定速度期望值 ⟨v⟩=d⟨x⟩dt\langle v \rangle = \frac{d\langle x \rangle}{dt}⟨v⟩=dtd⟨x⟩​，則動量期望值 ⟨p⟩=m⟨v⟩\langle p \rangle = m\langle v \rangle⟨p⟩=m⟨v⟩：

一般物理量的期望值

對於任何依賴於位置與動量的動力學量 Q(x,p)Q(x, p)Q(x,p)，其期望值的計算方法為將算符夾在波函數與其共軛複數之間：

⟨Q(x,p)⟩=∫Ψ∗Q(x,ℏi∂∂x)Ψ dx\langle Q(x, p) \rangle = \int \Psi^* Q\left(x, \frac{\hbar}{i} \frac{\partial}{\partial x}\right) \Psi \, dx \quad \text{} ⟨Q(x,p)⟩=∫Ψ∗Q(x,iℏ​∂x∂​)Ψdx

這個公式在量子力學中被視為一項公理或基本配方，是後續所有物理計算的基石。

海森堡不確定性原理 (The Uncertainty Principle)

粒子的位置與動量（透過波長 λ\lambdaλ 與德布羅意公式 p=h/λp = h/\lambdap=h/λ 聯繫）之間存在不可避免的權衡關係。這被量化為海森堡不確定性原理：

σxσp≥ℏ2\sigma_x \sigma_p \ge \frac{\hbar}{2} σx​σp​≥2ℏ​

這代表越精確地確定粒子的位置，其動量就越不確定，反之亦然。這並非測量儀器的缺點，而是波的本質屬性。

  • ⟨x2⟩\langle x^2 \rangle⟨x2⟩：利用公式 ∫−∞∞x2e−λx2dx=12λπλ\int_{-\infty}^{\infty} x^2 e^{-\lambda x^2} dx = \frac{1}{2\lambda}\sqrt{\frac{\pi}{\lambda}}∫−∞∞​x2e−λx2dx=2λ1​λπ​​，且已知 A2πλ=1A^2\sqrt{\frac{\pi}{\lambda}}=1A2λπ​​=1：

⟨x2⟩=∫−∞∞x2∣Ψ∣2 dx=12λ=12(2am/ℏ)=ℏ4am\langle x^2 \rangle = \int_{-\infty}^{\infty} x^2 |\Psi|^2 \, dx = \frac{1}{2\lambda} = \frac{1}{2(2am/\hbar)} = \frac{\hbar}{4am} ⟨x2⟩=∫−∞∞​x2∣Ψ∣2dx=2λ1​=2(2am/ℏ)1​=4amℏ​

  • ⟨p⟩\langle p \rangle⟨p⟩

⟨p⟩=md⟨x⟩dt\langle p \rangle = m \frac{d\langle x \rangle}{dt} ⟨p⟩=mdtd⟨x⟩​

因為 ⟨x⟩=0\langle x \rangle = 0⟨x⟩=0 為常數，故：

⟨p⟩=0\langle p \rangle = 0 ⟨p⟩=0

  • ⟨p2⟩\langle p^2 \rangle⟨p2⟩：利用 p2p^2p2 的期望值定義：

⟨p2⟩=∫Ψ∗(−ℏ2∂2∂x2)Ψ dx\langle p^2 \rangle = \int \Psi^* \left( -\hbar^2 \frac{\partial^2}{\partial x^2} \right) \Psi \, dx ⟨p2⟩=∫Ψ∗(−ℏ2∂x2∂2​)Ψdx

或者利用更簡便的關係，由於 ∣∂Ψ∂x∣2=(2amxℏ)2∣Ψ∣2|\frac{\partial \Psi}{\partial x}|^2 = (\frac{2amx}{\hbar})^2 |\Psi|^2∣∂x∂Ψ​∣2=(ℏ2amx​)2∣Ψ∣2：

⟨p2⟩=∫∣−iℏ∂Ψ∂x∣2dx=ℏ2∫4a2m2x2ℏ2∣Ψ∣2dx\langle p^2 \rangle = \int | -i\hbar \frac{\partial \Psi}{\partial x} |^2 dx = \hbar^2 \int \frac{4a^2m^2x^2}{\hbar^2} |\Psi|^2 dx ⟨p2⟩=∫∣−iℏ∂x∂Ψ​∣2dx=ℏ2∫ℏ24a2m2x2​∣Ψ∣2dx

⟨p2⟩=4a2m2⟨x2⟩=4a2m2(ℏ4am)=amℏ\langle p^2 \rangle = 4a^2m^2 \langle x^2 \rangle = 4a^2m^2 \left( \frac{\hbar}{4am} \right) = am\hbar ⟨p2⟩=4a2m2⟨x2⟩=4a2m2(4amℏ​)=amℏ

總結:

  • ⟨x⟩=0\langle x \rangle = 0⟨x⟩=0
  • ⟨x2⟩=ℏ4am\langle x^2 \rangle = \frac{\hbar}{4am}⟨x2⟩=4amℏ​
  • ⟨p⟩=0\langle p \rangle = 0⟨p⟩=0
  • ⟨p2⟩=amℏ\langle p^2 \rangle = am\hbar⟨p2⟩=amℏ

4. 不確定性原理 (Uncertainty Principle)

計算標準差 σ=⟨Q2⟩−⟨Q⟩2\sigma = \sqrt{\langle Q^2 \rangle - \langle Q \rangle^2}σ=⟨Q2⟩−⟨Q⟩2​。

  • 計算 σx\sigma_xσx​ 和 σp\sigma_pσp​:

σx=ℏ4am−02=ℏ4am\sigma_x = \sqrt{\frac{\hbar}{4am} - 0^2} = \sqrt{\frac{\hbar}{4am}} σx​=4amℏ​−02​=4amℏ​​

σp=amℏ−02=amℏ\sigma_p = \sqrt{am\hbar - 0^2} = \sqrt{am\hbar} σp​=amℏ−02​=amℏ​

  • 驗證乘積:

σxσp=ℏ4am⋅amℏ=ℏ⋅amℏ4am=ℏ24=ℏ2\sigma_x \sigma_p = \sqrt{\frac{\hbar}{4am}} \cdot \sqrt{am\hbar} = \sqrt{\frac{\hbar \cdot am\hbar}{4am}} = \sqrt{\frac{\hbar^2}{4}} = \frac{\hbar}{2} σx​σp​=4amℏ​​⋅amℏ​=4amℏ⋅amℏ​​=4ℏ2​​=2ℏ​

結論：海森堡不確定性原理指出 σxσp≥ℏ2\sigma_x \sigma_p \geq \frac{\hbar}{2}σx​σp​≥2ℏ​。

此波函數的乘積恰好等於 ℏ2\frac{\hbar}{2}2ℏ​，符合不確定性原理（這是高斯波包的特性，即最小不確定性態）。
