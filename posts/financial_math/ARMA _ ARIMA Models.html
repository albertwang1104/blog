ARMA & ARIMA Models

時間序列分析（Time Series Analysis）主要研究隨時間變化的資料結構、性質及其預測方法。數學上，我們通常考慮一個定義在離散時間 t∈Zt \in \mathbb{Z}t∈Z 上的隨機過程 {Xt}\{X_t\}{Xt​}，並探討其統計性質、動態結構及模型建構。

Definition

時間序列（Time Series)

A stochastic process {Xt:t∈Z}\{X_t : t \in \mathbb{Z}\}{Xt​:t∈Z} consisting of random variables indexed by time index t is a time series.

隨機過程 (stochastic process)

考慮一列隨機變數 {Xt:t∈Z}\{X_t : t \in \mathbb{Z}\}{Xt​:t∈Z}，每個 XtX_tXt​ 都定義在某個機率空間 (Ω,F,P)(\Omega, \mathcal{F}, P)(Ω,F,P) 上。此過程的聯合分布為

P(Xt1∈A1, Xt2∈A2, …, Xtk∈Ak),∀ t1,t2,…,tk∈Z.P\left(X_{t_1} \in A_1, \, X_{t_2} \in A_2, \, \dots, \, X_{t_k} \in A_k\right), \quad \forall\, t_1, t_2, \dots, t_k \in \mathbb{Z}. P(Xt1​​∈A1​,Xt2​​∈A2​,…,Xtk​​∈Ak​),∀t1​,t2​,…,tk​∈Z.

That is all finite-dimensional distributions of {Xt}\{X_t\}{Xt​}.

強平穩（Strict Stationarity）

{Xt}\{X_t\}{Xt​} is Strictly Stationary if ∀τ∈Z\forall \tau \in \mathbb{Z}∀τ∈Z, ∀t1,t2,…,tk\forall t_1, t_2, \dots, t_k∀t1​,t2​,…,tk​, ∀k\forall k∀k

P(Xt1+τ, Xt2+τ, …, Xtk+τ)=P(Xt1, Xt2, …, Xtk)P\left(X_{t_1+ \tau}, \, X_{t_2+ \tau} , \, \dots, \, X_{t_k+ \tau}\right) = P\left(X_{t_1}, \, X_{t_2} , \, \dots, \, X_{t_k} \right) P(Xt1​+τ​,Xt2​+τ​,…,Xtk​+τ​)=P(Xt1​​,Xt2​​,…,Xtk​​)

or

(Xt1,Xt2,…,Xtk)=d(Xt1+h,Xt2+h,…,Xtk+h).(X_{t_1}, X_{t_2}, \dots, X_{t_k}) \overset{d}{=} (X_{t_1+h}, X_{t_2+h}, \dots, X_{t_k+h}). (Xt1​​,Xt2​​,…,Xtk​​)=d(Xt1​+h​,Xt2​+h​,…,Xtk​+h​).

弱平穩（Covariance Stationarity / Second-Order Stationarity）

A time series {Xt}\{X_t\}{Xt​} is Covariance Stationary if

  • E[Xt]=μ\mathbb{E}[X_t] = \muE[Xt​]=μ is constant for all time t.
  • Var⁡(Xt)=σX2\operatorname{Var}(X_t) = \sigma^2_XVar(Xt​)=σX2​
  • Cov⁡(Xt,Xt+τ)=γ(τ)\operatorname{Cov}(X_t,X_{t+\tau})=\gamma(\tau)Cov(Xt​,Xt+τ​)=γ(τ), 表示 covariance 只跟時間差有關。
    The auto-correlation function of {Xt}\{X_t\}{Xt​} is defined by

    ρ(τ)=Cov⁡(Xt,Xt+τ)Var⁡(Xt)Var⁡(Xt+τ)=γ(τ)γ(0)\rho (\tau) = \frac{\operatorname{Cov}(X_t,X_{t+\tau})}{\sqrt{\operatorname{Var}(X_t)\operatorname{Var}(X_{t+\tau})}} = \frac{\gamma(\tau)}{\gamma (0)} ρ(τ)=Var(Xt​)Var(Xt+τ​)​Cov(Xt​,Xt+τ​)​=γ(0)γ(τ)​

Wold Representation Theorem

Wold Representation Theorem 提供了任何零均值、平穩過程的一個重要分解形式。

Wold Representation Theorem

Any zero-mean covariance stationary time series {Xt}\{X_t\}{Xt​} and ∑h=−∞∞∣γ(h)∣<∞\sum_{h=-\infty}^{\infty} |\gamma(h)| < \infty∑h=−∞∞​∣γ(h)∣<∞ can be decomposed as

Xt=St+Vt=∑j=0∞ψjϵt−j+Vt,∀ t,X_t = S_t + V_t = \sum_{j=0}^{\infty} \psi_j \epsilon_{t-j} + V_t, \quad \forall\, t, Xt​=St​+Vt​=j=0∑∞​ψj​ϵt−j​+Vt​,∀t,

where

  • {Vt}\{V_t\}{Vt​} is a linearly deterministic process, i.e., a linear combination of past values of {Vt}\{V_t\}{Vt​} with constant coeﬃcients.
  • St=∑j=0∞ψjϵt−jS_t = \sum_{j=0}^{\infty} \psi_j \epsilon_{t-j}St​=∑j=0∞​ψj​ϵt−j​ is an infinite moving average process of error terms, where
    • ∑j=0∞ψj2<∞, ψ0=1\sum_{j=0}^{\infty} \psi_j^2 < \infty, \, \psi_0 = 1∑j=0∞​ψj2​<∞,ψ0​=1
    • {ϵt}\{\epsilon_t\}{ϵt​} is linearly unpredictable white noise, i.e. $$\mathbb{E}[\epsilon_t] = 0,\quad \operatorname{Var}(\epsilon_t) = \sigma^2_\epsilon, \quad \mathbb{E}[\epsilon_s\epsilon_t] = 0, \forall s \neq t$$
    • {ϵt}\{\epsilon_t\}{ϵt​} is uncorrelated with {Vt}\{V_t\}{Vt​} i.e. E[ϵtVt]=0\mathbb{E}[\epsilon_tV_t] = 0E[ϵt​Vt​]=0.

這個定理表示任何平穩過程均可分解為一個移動平均項 VtV_tVt​ 以及一個根據過去的值預測出來的部分 StS_tSt​。

Lag Operator

Lag Operator (延遲算子) 是時間序列分析中的一個重要工具，用於簡化滯後項的表示。

定義

Lag Operator LLL 是一種線性算子，其作用是將時間序列的索引延遲（或滯後）一個單位時間。對於一個時間序列 {Xt}\{X_t\}{Xt​}，滯後算子的作用定義為：

L(Xt)=Xt−1.L(X_t) = X_{t-1}. L(Xt​)=Xt−1​.

性質

  1. 高階滯後：
    滯後算子可以疊加使用，表達多步滯後。對於任意正整數 kkk，

    Lk(Xt)=Xt−k.L^k(X_t) = X_{t-k}. Lk(Xt​)=Xt−k​.

    Inverses of these operators are well defined as:

    L−k(Xt)=Xt+k.L^{-k}(X_t) = X_{t+k}. L−k(Xt​)=Xt+k​.

  2. 恆等性：

    L0(Xt)=Xt.L^0(X_t) = X_t. L0(Xt​)=Xt​.

  3. 線性性：
    滯後算子對於常數加權和具有線性性質。對於兩個時間序列 {Xt}\{X_t\}{Xt​} 和 {Yt}\{Y_t\}{Yt​}，以及常數 aaa 和 bbb，有：

    L(aXt+bYt)=aL(Xt)+bL(Yt).L(aX_t + bY_t) = aL(X_t) + bL(Y_t). L(aXt​+bYt​)=aL(Xt​)+bL(Yt​).

ARMA Models

自回歸移動平均（Autoregressive Moving Average, ARMA）模型是時序分析中常用來刻畫平穩過程動態行為的模型。一般的 ARMA (p,q)(p,q)(p,q) 模型定義為：

Xt=ϕ1Xt−1+ϕ2Xt−2+⋯+ϕpXt−p+ϵt+θ1ϵt−1+θ2ϵt−2+⋯+θqϵt−q,X_t = \phi_1 X_{t-1} + \phi_2 X_{t-2} + \cdots + \phi_p X_{t-p} + \epsilon_t + \theta_1 \epsilon_{t-1} + \theta_2 \epsilon_{t-2} + \cdots + \theta_q \epsilon_{t-q}, Xt​=ϕ1​Xt−1​+ϕ2​Xt−2​+⋯+ϕp​Xt−p​+ϵt​+θ1​ϵt−1​+θ2​ϵt−2​+⋯+θq​ϵt−q​,

或用 lag operator 表示：

ϕ(L)Xt=θ(L)ϵt,\phi(L) X_t = \theta(L) \epsilon_t, ϕ(L)Xt​=θ(L)ϵt​,

其中：

  • ϕ(L)=1−ϕ1L−ϕ2L2−⋯−ϕpLp\phi(L) = 1 - \phi_1 L - \phi_2 L^2 - \cdots - \phi_p L^pϕ(L)=1−ϕ1​L−ϕ2​L2−⋯−ϕp​Lp 為自回歸（AR）部分；
  • θ(L)=1+θ1L+θ2L2+⋯+θqLq\theta(L) = 1 + \theta_1 L + \theta_2 L^2 + \cdots + \theta_q L^qθ(L)=1+θ1​L+θ2​L2+⋯+θq​Lq 為移動平均（MA）部分；
  • {ϵt}\{\epsilon_t\}{ϵt​} 為白雜訊序列，ϵt∼WN(0,σ2)\epsilon_t \sim WN(0, \sigma^2)ϵt​∼WN(0,σ2)。

條件與性質：

  • 平穩性條件：對 AR 部分，要求多項式 ϕ(z)=0\phi(z)=0ϕ(z)=0 的根均在單位圓外（即 ∣z∣>1|z|>1∣z∣>1）。
  • 可逆性條件：對 MA 部分，要求多項式 θ(z)=0\theta(z)=0θ(z)=0 的根均在單位圓外。

在符合上述條件下，ARMA 模型可以看作是 Wold 表示定理的有限參數化特例。

ARIMA Model

ARIMA 模型（Autoregressive Integrated Moving Average）是 ARMA 模型的推廣，適用於非平穩時序資料。其基本想法是透過差分運算將非平穩序列轉換為平穩序列，再建立 ARMA 模型。

設原始序列為 {Yt}\{Y_t\}{Yt​}，若經過 ddd 次差分後得到平穩序列 {Xt}\{X_t\}{Xt​}：

Xt=ΔdYt,X_t = \Delta^d Y_t, Xt​=ΔdYt​,

其中差分算子定義為

ΔYt=Yt−Yt−1=Yt−L(Yt)=(1−L)(Yt)=ΔYt,Δd=(1−L)d.\Delta Y_t = Y_t - Y_{t-1} = Y_t - L(Y_t) = (1-L)(Y_t) = \Delta Y_t, \quad \Delta^d = (1-L)^d. ΔYt​=Yt​−Yt−1​=Yt​−L(Yt​)=(1−L)(Yt​)=ΔYt​,Δd=(1−L)d.

則 ARIMA (p,d,q)(p,d,q)(p,d,q) 模型可表示為：

ϕ(L)ΔdYt=θ(L)ϵt,\phi(L) \Delta^d Y_t = \theta(L) \epsilon_t, ϕ(L)ΔdYt​=θ(L)ϵt​,

其中：

  • ϕ(L)\phi(L)ϕ(L) 與 θ(L)\theta(L)θ(L) 定義同 ARMA 模型；
  • ddd 為整數，表示差分的次數，使得 ΔdYt\Delta^d Y_tΔdYt​ 為平穩序列。

Testing for Stationarity/Non-Stationarity

在實際應用中，判斷資料是否平穩是建模的前提之一。常用的檢驗方法包括：

  1. 單位根檢定（Unit Root Test）：

    • Augmented Dickey-Fuller (ADF) 檢定：
      ADF 檢定檢驗虛無假設 H0H_0H0​ 為序列存在單位根（即非平穩），其基本迴歸式為

      ΔXt=α+βt+γXt−1+∑i=1kδiΔXt−i+ϵt.\Delta X_t = \alpha + \beta t + \gamma X_{t-1} + \sum_{i=1}^{k} \delta_i \Delta X_{t-i} + \epsilon_t. ΔXt​=α+βt+γXt−1​+i=1∑k​δi​ΔXt−i​+ϵt​.

      此處，若 γ=0\gamma = 0γ=0 則表示單位根存在。檢驗統計量為 γ^/SE⁡(γ^)\hat{\gamma}/\operatorname{SE}(\hat{\gamma})γ^​/SE(γ^​)，並與相應的臨界值比較。

    • Phillips-Perron (PP) 檢定：
      類似於 ADF，但對序列的自相關與異方差做出不同修正，使用非參數方法修正檢定統計量。

  2. 平穩性檢定：

    • KPSS 檢定（Kwiatkowski-Phillips-Schmidt-Shin）：
      與單位根檢定相反，KPSS 檢定的虛無假設 H0H_0H0​ 為序列平穩。檢定基於序列殘差的累積和與長期變異數的比值，檢驗統計量通常形式為

      η=1T2∑t=1TSt2/σ^2,\eta = \frac{1}{T^2} \sum_{t=1}^{T} S_t^2 \bigg/ \hat{\sigma}^2, η=T21​t=1∑T​St2​/σ^2,

      其中 St=∑i=1tϵ^iS_t = \sum_{i=1}^{t} \hat{\epsilon}_iSt​=∑i=1t​ϵ^i​ 為累積和，σ^2\hat{\sigma}^2σ^2 為長期變異數的估計。

  3. 其他圖形與自相關分析：

    • ACF/PACF 分析：自相關函數（ACF）與偏自相關函數（PACF）的圖形檢查，可提供是否存在趨勢或週期性結構的直觀證據。
    • 結構變化檢定：例如 Chow 檢定，可用來檢查時序資料中是否存在結構性轉變，這也是非平穩性的可能來源之一。

綜上，通過上述檢定方法，可以確定時序資料是否平穩，從而指導後續模型的建立（例如是否需要差分轉換）。

結論

  • Two-Step Regression Estimation：基於 OLS，計算簡單但準確度較低。
  • Yule-Walker Estimation：適用於 AR 模型，但對 MA 組件較弱。
  • MLE：在樣本量足夠大時最有效，並具有最佳的統計性質，但計算量較大。
