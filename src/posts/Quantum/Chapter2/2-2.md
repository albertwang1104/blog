無窮深位能井（The Infinite Square Well）是一個理想化的模型，在物理學中是一個極為經典的例子。

## 位能分布與物理模型
假設位能函數 $V(x)$ 的形式如下：
$$V(x) = \begin{cases} 0, & \text{if } 0 \le x \le a \\ \infty, & \text{otherwise} \end{cases}$$
在這個位能分布中，粒子在井內（$0$ 到 $a$ 之間）是完全自由的，但在井的兩端（$x=0$ 與 $x=a$）會受到**無限大的力**阻止其逃逸。

## 求解薛丁格方程式

在井外，由於位能無限大，波函數 $\psi(x)$ 必須等於 $0$。而在井內（$V=0$），不隨時間演化的薛丁格方程式簡化為：

$$-\frac{\hbar^2}{2m} \frac{d^2\psi}{dx^2} = E\psi \quad \text{or} \quad \frac{d^2\psi}{dx^2} = -k^2\psi, \quad \text{where } k \equiv \frac{\sqrt{2mE}}{\hbar}$$

這是一個古典的**簡諧振蕩方程式**，其一般解為：
$$\psi(x) = A \sin kx + B \cos kx$$

### 邊界條件與能量量子化
根據波函數的連續性，$\psi(x)$ 在井壁邊界必須與井外的 $0$ 相接：
- **在 $x=0$ 時**：$\psi(0) = A \sin(0) + B \cos(0) = B = 0$。因此 $B$ 必須為 $0$，解簡化為 $\psi(x) = A \sin kx$。
- **在 $x=a$ 時**：$\psi(a) = A \sin ka = 0$。由於 $A=0$ 會導致無物理意義的零解，因此必須要求 $ka$ 為 $\pi$ 的整數倍：
    $$ka = n\pi, \quad  n = 1, 2, 3, \dots$$

又 $k = \frac{\sqrt{2mE}}{\hbar}$，由此導出**能量的量子化**：
$$E_n = \frac{n^2 \pi^2 \hbar^2}{2ma^2}$$

### 歸一化與定態波函數

透過**歸一化條件** $\int_0^a |\psi|^2 dx = 1$，求得常數 $A = \sqrt{2/a}$。因此，無限深位能井的 **定態波函數（Stationary States）** 為：

$$\psi_n(x) = \sqrt{\frac{2}{a}} \sin\left(\frac{n\pi x}{a}\right)$$

## 重要性質與物理特徵

1.  **奇偶對稱性**：波函數相對於井中心（$x=\frac{a}{2}$）交替呈現偶對稱（Even）與奇對稱（Odd），即正弦函數的對稱性。
![image](img/infinite%20wall%20energy.png)
2.  **節點（Nodes）**：隨著能量增加，波函數的節點數也隨之增加（$\psi_n$ 有 $n-1$ 個節點）。而能量最低（$n=1$）的定態波函數稱為**基態（grond state）**，其他（$n \geq 2$）定態波函數能量成平方成長，都稱為**激發態（excited states）**。
3.  **正交歸一性（Orthonormality）**：不同能階的波函數彼此正交，滿足 $\int \psi_m^* \psi_n dx = \delta_{mn}$。其中 $\delta_{mn}$ 是 Kronecker delta（$m=n$ 時為 1，$m \neq n$ 時為 0）。
4. **完備性（Completeness）**：任何井內的波函數 $f(x)$ 都可以表示為這些定態的線性組合。
$$f(x) = \sum_{n=1}^{\infty} c_n \psi_n(x) = \sqrt{\frac{2}{a}} \sum_{n=1}^{\infty} c_n \sin\left(\frac{n\pi}{a}x\right)$$
這本質上就是 $f(x)$ 的傅立葉級數。

### 一般隨時間演化的解（通解）

系統最一般的解是各個定態隨時間演化的線性疊加：
$$\Psi(x, t) = \sum_{n=1}^{\infty} c_n \sqrt{\frac{2}{a}} \sin\left(\frac{n\pi x}{a}\right) e^{-iE_n t/\hbar}, \quad E_n = \frac{n^2 \pi^2 \hbar^2}{2ma^2}$$

在給定初始波函數 $\Psi_0 = \Psi(x, 0)$ 的條件下，我們可以計算擴展係數 $c_n$：

$$c_n = \langle \psi_n | \Psi_0 \rangle = \sqrt{\frac{2}{a}} \int_0^a \sin\left(\frac{n\pi}{a}x\right) \Psi(x, 0) dx $$


#### 比喻：

你可以把無限深位能井中的粒子想像成**被固定在兩端的吉他弦**。弦只能以特定的頻率（能量）振動，而這些振動模式（波函數）決定了粒子在井中各處出現的機率。就像你無法在吉他弦上彈出任意頻率一樣，量子井中的粒子也只能擁有特定的、不連續的能量能階。
